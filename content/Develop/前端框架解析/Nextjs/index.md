> Nextjs 是一个全栈开发框架，可以做服务端也可以做客户端， 
# 基础

## 安装

```bash
npx create-next-app@latest
```

## 启动

```bash
 npm run dev
```

访问： http://localhost:3002

## 查看安装的版本号

```bash
grep '"next":' package.json
```

> 示例输出：    
> "next": "^14.0.4",
