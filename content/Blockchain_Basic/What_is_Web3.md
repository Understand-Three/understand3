>[!NOTE]
>- Web 1.0 就是关于获取和读取信息。 
>- Web 2.0 是如今普遍的互联网形态，它是关于读取、写入、创建以及与最终用户交互的。它被称为参与式社交网络。
>- Web 3.0 是正在探索的第三代万维网，是去中心化网络的愿景。这一切都与阅读、写作和拥有有关。

> 你可能听多数人说 Web3 和 Web3.0 是两个不同的东西。在这里，我不想搞得太复杂，所以本文的 Web3 和 Web3.0 是通用的，都是基于区块链技术的应用。关于他们的区别，会在文末简单说明


# 互联网

## Web 1.0

一开始，由电线和服务器组成的物理基础设施，让计算机和它们前面的人可以相互交谈。美国政府的 [ARPANET](https://zh.wikipedia.org/wiki/ARPANET) 于 1969 年发送了第一条消息，但我们今天所知道的 Web 直到 1991 年才出现，当时 [HTML](https://zh.wikipedia.org/wiki/HTML) 和 [URL](https://zh.wikipedia.org/wiki/%E7%BB%9F%E4%B8%80%E8%B5%84%E6%BA%90%E5%AE%9A%E4%BD%8D%E7%AC%A6) 使用户能够在静态页面之间导航。将此视为只读 Web 或 Web1.0。

在 Web 1.0 中，禁止在浏览 Internet 时在网站上投放广告。此外，在 Web 1.0 中，**Ofoto** 是一个在线数码摄影网站，用户可以在其中存储、共享、查看和打印数字图像。Web 1.0 是一个**内容交付网络（CDN）**，允许在网站上显示信息。它可以用作个人网站。它根据查看的页面对用户进行成本计算。它具有允许用户检索特定信息的目录。Web 1.0 时代大约从 1991 年持续到 2004 年。 



### Web 1.0 网站的四个设计要素包括：

1. 静态页面。

2. 内容从服务器的文件系统提供。

3. 使用服务器端包含或通用网关接口 （CGI） 创建的页面。

4. 框架和表格用于定位和对齐页面上的元素。

### Web 1.0 的特点

- 通过超链接轻松将静态页面与系统连接

- 使用 HTML 3.2 支持框架和表格等元素

- 还有图形和一个 GIF 按钮

- 减少用户和服务器之间的交互

- 您可以 通过电子邮件发送 HTML 表单

- 仅提供单向发布媒体



## Web 2.0

在 2000 年代初期，情况开始发生变化。首先，互联网的互动性越来越强;这是一个用户生成内容或读/写 Web 的时代。社交媒体是 Web2（或 Web 2.0）的一个关键功能，Facebook（Meta）、Twitter 和 Tumblr 开始定义在线体验。YouTube、Wikipedia 和 Google 以及对内容发表评论的能力，扩展了我们观看、学习、搜索和交流的能力。

[Web2.0](https://zh.wikipedia.org/wiki/Web_2.0) 时代也是中心化时代。网络效应和规模经济带来了明显的赢家，这些公司（其中许多在上面列出）通过抓取用户数据并出售针对这些数据的定向广告，为自己和股东创造了令人难以置信的财富。这使得服务可以“免费”提供，尽管用户最初并不理解这种交易的含义。Web2 还为普通人创造了新的赚钱方式，例如通过共享经济和网红工作。

在当前的系统中有很多值得批评的地方：拥有集中或近乎垄断权力的公司往往未能负责任地使用它，现在意识到自己就是产品的消费者对放弃对个人数据的控制权越来越不舒服，而且目标广告经济有可能是一个脆弱的泡沫，对实际提振广告商几乎没有作用。随着 Web 的发展、中心化和公司化，许多人开始怀疑当前的互联网环境是否有更好的未来。

2004 年 当 Web 2.0 一词因 Tim O'Reilly 和 Dale Dougherty 举办的第一届 Web 2.0 会议（后来称为 Web 2.0 峰会）而出名时，该术语由 Darcy DiNucci 于 1999 年创造。Web 2.0 是指为最终用户突出用户生成的内容、可用性和互操作性的全球网站。Web 2.0 也称为参与式社交网络。它不是指对任何技术规范的修改，而是指修改网页的设计和使用方式。这种转变是有益的，但当变化发生时似乎并非如此。Web 2.0 作为虚拟社区中用户生成内容的创建者，允许在社交媒体对话中相互交互和协作。Web 2.0 是 Web 1.0 的增强版本。



### Web 2.0 功能

1. 自由排序信息，允许用户集体检索和分类信息。

2. 响应用户输入的动态内容。

3. 信息通过在线评级和评论在网站所有者和网站用户之间流动。

4. 为实现自用而开发的 API，例如通过软件应用程序。

5. 对于访问 Web 存在不同的关注点，从传统的 Internet 用户群到更广泛的用户。

### 使用 Web 2.0

社交网络包含多个在线工具和平台，人们可以在其中分享他们的观点、意见、想法和经验。Web 2.0 应用程序往往与最终用户的交互要多得多。因此，最终用户不仅是应用程序的用户，而且还是下面提到的 8 个工具的参与者：

1. 播客
2. 写博客
3. 标记
4. 使用 RSS 进行管理
5. 为社交添加书签
6. 社交媒体
7. 社交媒体
8. Web 内容投票



## Web3

这就把我们带到了 Web3。这一愿景的倡导者将其宣传为一次根深蒂固的更新，将纠正 Web2 的问题和不正当的激励措施。担心隐私问题？加密钱包保护您的在线身份。关于审查制度？去中心化数据库以不可变和透明的方式存储所有内容，防止版主突然进入删除违规内容。集中？您可以对您花费时间的网络做出的决策进行真正的投票。不仅如此，您还会获得_有价值的_股份 — 您不是产品，而是所有者。这就是读 / 写 / 自己的 Web 的愿景。

1991 年，科学家 W. Scott Stornetta 和 Stuart Haber 推出了第一个区块链 —— 一个为数字文档添加时间戳的项目，从而为 Web3 埋下了种子。但这个想法直到 2009 年才真正扎根，当时[比特币](https://www.ussc.gov/sites/default/files/pdf/training/annual-national-training-seminar/2018/Emerging_Tech_Bitcoin_Crypto.pdf)是在金融危机之后由化名发明者中本聪推出的（至少部分是为了应对金融危机）。它及其基础区块链技术的工作原理是这样的：加密货币的所有权在共享的公共分类账上进行跟踪，当一个用户想要进行转账时，“矿工”通过解决复杂的数学问题来处理交易，向链中添加一个新的数据“块”，并通过他们的努力赚取新创建的比特币。虽然比特币链仅用于货币，但较新的区块链提供了其他选择。以太坊于 2015 年推出，既是一种加密货币，也是一个可用于构建其他加密货币和区块链项目的平台。其联合创始人之一加文·伍德 （Gavin Wood） 将[以太坊](https://www.newstatesman.com/science-tech/2016/04/ethereum-competitor-bitcoin-which-could-transform-entire-industries)描述为“一台适用于整个地球的计算机”，计算能力分布在全球，无处可控。现在，十多年后，基于区块链的网络的支持者宣布一个新时代 —— Web3 —— 已经到来。

简而言之，Web3 是加密货币的延伸，以新的方式使用区块链来实现新的目的。区块链可以存储钱包中的代币数量、自动执行合同的条款或去中心化应用程序 （dApp） 的代码。并非所有区块链都以相同的方式工作，但一般来说，硬币被用作激励矿工处理交易的动力。在比特币这样的“工作量证明”链上，解决处理交易所需的复杂数学问题在设计上是能源密集型的。在较新但越来越普遍的“权益证明”链上，处理交易只需要在链中拥有权益的验证者同意交易是合法的——这个过程要高效得多。在这两种情况下，交易数据都是公开的，尽管用户的钱包只能通过加密生成的地址来识别。区块链是“只写”的，这意味着您可以向其添加数据，但不能删除数据。

> [!NOTE] 关于 `web3` 和 `web3.0` 的区别
>
> **Web3.0**
>
> 数字加点的形式，例如 `1.0`、 `2.0` 一般表示版本，所以 `Web1.0` 和 `Web2.0` 表示的是基于万维网（World Wide Web）或者因特网（Internet）的技术革新，所以一些人认为 `Web3.0` 也应该是一种基于万维网或者因特网的技术革新，所以他们认为 `Web3.0` 是语义网（Semantic Web，3.0），即一种比 Google 现有的引擎模式所能达到的更合理的方式建立“世界信息”。
>
> 从机器概念的角度来看，与人类的理解相比，这一点尤其正确。语义网需要使用像 OWL 这样的声明性本体论语言来生成特定于领域的本体论，机器可以使用它来推理信息并得出新的结论，而不仅仅是匹配关键字，至于其他的特征技术，例如：人工智能、可访问性和全球性，则是实现语义网的技术基础，以在用户使用自然语言的情况下获取精准的内容匹配。
>
> `Web3.0` 的应用：语言助手、个性化推荐引擎、语义搜索引擎
>
> 
>
> **Web3**
>
> 而 `Web3` 则是一种创新技术，主要是基于区块链技术的去中化网络。`Web3` 并不需要万维网或者因特网，只是在目前的阶段，不得已使用万维网或者互联网访问区块链的特定信息。
>
> `Web3` 实际上是基于区块链的一种技术应用，区块链是一种分布式账本系统，用于以安全和透明的方式记录交易。这使得 Web3 非常适合金融交易和[供应链管理](https://cointelegraph.com/explained/how-blockchain-technology-is-used-in-supply-chain-management)等应用。在区块链之上的智能合约和密码学等技术被用来保障用户的隐私安全，并衍生了系列金融、游戏和社交等方面的创新应用。
>
> `Web3` 的应用：去中心化金融（DeFi）、非同质化代币（NFT）、去中心化自治组织（DAO）
>
> 一言以蔽之，`Web3` 专注于去中心化和安全性，而 `Web 3.0` 专注于智能和连接性。`Web 3.0` 设想了一个高度智能的 Web，它使机器能够理解和解释信息，从而促进各种应用程序和服务之间的无缝通信。
>
> 目前多数人认为他们是不同的东西，事实上，就目前而言它们确实是不同的东西。但是它们仍然在积极探索中，未来充满了不确定性，随着去中心化技术变得更加智能，智能应用程序变得更加去中心化，它们也许最终会融合在一起。
>
> 所以 `Web3` 和 `Web3.0`，在一般意义上都是指的区块链技术的应用，而人工智能、推荐引擎等语义网一般被认为是现阶段的技术创新，这个含糊不清的阶段至少要持续到 `Web3.0` 中出现巨型企业后。
